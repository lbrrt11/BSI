Uma mensagem é criptografada invertendo cada palavra do texto. O texto “Uma mensagem confidencial” criptografado fica “amU megasnem laicnedifnoc”. Implemente a criptografia e a decriptografia de mensagem. Faça isso utilizando Pilha. Para pegar caracteres específicos de uma String, utilize seu método charAt(int). Quando for colocar o caractere na Pilha, você vai perceber que não pode usar tipos primitivos (char) como tipo parametrizado, então em vez de declarar uma pilha de char crie uma pilha do tipo wrapper Character.
